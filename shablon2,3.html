Задание ЕГЭ №4


























Для кодирования некоторой последовательности, состоящей из букв А, Б, В, Г, Д, Е, решили использовать неравномерный двоичный код, удовлетворяющий условию Фано. Для букв А, Б, В, Г использовали соответственно кодовые слова 000, 001, 10, 11.
Укажите кратчайшее возможное кодовое слово для буквы Д, при котором код будет допускать однозначное декодирование. Если таких кодов несколько, укажите код с наименьшим числовым значением.

Построим дерево для заданных кодовых слов(А-000, Б-001, В-10, Г-11). Может показаться, что ответ-01, поскольку на эту ветвь можно "прикрепить" букву Д, но это не так- тогда некуда будет "прикреплять" букву Е, поэтому для того, чтобы добавить в это дерево буквы (Д и Е) и сохранить выполнение условия Фано, нужно в узле 01 сделать развилку, тогда получается 2 свободных кода: 010 и 011, из них меньший-010
Ответ: 010
