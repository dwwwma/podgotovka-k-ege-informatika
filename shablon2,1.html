Задание ЕГЭ №4


























По каналу связи передаются сообщения, содержащие только четыре буквы: П, О, С, Т; для передачи используется двоичный код, допускающий однозначное декодирование. Для букв Т, О, П используются такие кодовые слова: Т: 111, О: 0, П:100.
Укажите кратчайшее кодовое слово для буквы С, при котором код будет допускать однозначное декодирование. Если таких кодов несколько, укажите код с наименьшим числовым значением.

Построим дерево дял заданных кодовых слов(О-0, Т-111,П-100). Для буквы "С" остались 2 пустые ячейки, на которые можно будт её "прикрепить" - это 101 И 110. Но так как нам нужно выбрать кодовое слово с наименьшим числовым значением, то мы выбираем "101"
Ответ: 101
